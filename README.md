# spotify-app-sf
test app


## Getting Starter

```sh
git clone https://github.com/fagurto/spotify-app-sf.git
cd spotify-app-sf

```
## Starting api server
```sh
cd backend/spotify-api/
mvn install
mvn package
java -jar target/spotify-api-0.0.1-SNAPSHOT.jar

```

## Starting frontend server
```sh
cd frontend/spotify-ui/
npm install
npm  start

```
